Dungeons & Dragons Character Creator

Concept

In this project, my group wanted to create a local application that would allow the user to create a Dungeons and Dragons character and display the character's information on a screen.

The biggest issue with this project was that we did not have any experience at the time with databases, so we ended up having to find a work-around for it. Our final version only allowed for a few races, classes, and backgrounds, but it ultimately had several nice components that were worth keeping.

This character creator has 4 character slots that can be used to store character information. The user can set up name, race, subrace, class, background, and alignment for their characters. They can also enter their character's stats along with proper modifiers for them. The last page in the app allows for the user to view the character's information and store it in the character slot of their choice.

Future Implementations

I want to recreate this application again in the future and by then I hope to have a better understanding of databases so that I could create a better character creator that will be a useful implementation for Dungeons and Dragons. I want the future of this app to help the player streamline the process of creating characters and will help them access all of their character's information. Ideally, this app will be the only thing that the player will need to use to play the game.

In order to do this, I will first need to create a databases that will contain all of the information that is available to D&D players. I will then need to use SQL to be able to manage the database and I will need to create the rest of the app in a way that will allow the user to make and portray their characters information. I know that I have to do before I could make a version of this app that i could actually be happy with, but I hope that at some point, I will be able to come back to this project and make a way better version of it.
